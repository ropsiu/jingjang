# jingjang


```bash
git clone https://github.com/ropsiu/jingjang.git
```

Make sure you have `bower` and `gulp` installed by testing:
```bash
bower --version
```

```bash
gulp --version
```

Enter the jingjang directory and run the build script:
```bash
cd jingjang && npm run build
```

Now you can run:
```bash
npm run start
```
